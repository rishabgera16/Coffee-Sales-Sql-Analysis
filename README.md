# Coffee Sales Data Analysis using PostgreSQL

## About the Project

This project focuses on analyzing a coffee sales dataset using PostgreSQL. The objective was to answer business-related questions by writing SQL queries and extracting meaningful insights from sales, customer, product, and city data.

Instead of simply retrieving data, the analysis aims to identify revenue trends, customer behavior, product performance, and city-wise business opportunities.

---

## Dataset

The project uses four tables:

• City
• Customers
• Products
• Sales

---

## Business Questions Solved

✔ Estimated coffee consumers by city

✔ Total revenue generated during Q4 2023

✔ Product-wise sales performance

✔ Average sales per customer

✔ Top-performing cities

✔ Customer purchasing behaviour

✔ Revenue contribution by city

✔ Additional business insights using joins, aggregations and filtering
